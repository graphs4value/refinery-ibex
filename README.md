# Refinery IBEX Java Bindings

[![Build](https://github.com/graphs4value/refinery-ibex/actions/workflows/build.yml/badge.svg)](https://github.com/graphs4value/refinery-ibex/actions/workflows/build.yml)

The repository contains Java bindings for
[IBEX](https://ibex-team.github.io/ibex-lib/) for use in the
[Refinery](https://refinery.tools) project.

Some code in this repository has been adapted from
[Google OR-TOOLS](https://developers.google.com/optimization/), which is
available under
[The Apache License, Version 2.0](http://www.apache.org/licenses/LICENSE-2.0.txt).

The artifacts produced in the repository compile upstream IBEX releases and package them as Maven modules.
